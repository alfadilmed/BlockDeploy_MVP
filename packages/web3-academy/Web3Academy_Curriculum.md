## 1. Introduction

Ce document définit le curriculum initial pour la Web3 Academy de BlockDeploy. L'objectif est de fournir une base solide de connaissances pour les nouveaux venus dans l'écosystème Web3 et pour les utilisateurs souhaitant maîtriser la plateforme BlockDeploy.

## 2. Philosophie du Curriculum

*   **Progressif :** Commencer par les concepts fondamentaux avant d'aborder des sujets plus complexes ou spécifiques à BlockDeploy.
*   **Pratique :** Intégrer des exemples concrets et des tutoriels pas-à-pas, surtout pour les cours liés à BlockDeploy.
*   **Accessible :** Utiliser un langage clair et éviter le jargon excessif, ou l'expliquer lorsqu'il est inévitable.
*   **Modulaire :** Permettre aux utilisateurs de choisir les cours ou modules qui les intéressent le plus.
*   **Orienté BlockDeploy :** Mettre en évidence comment BlockDeploy simplifie les processus Web3.

## 3. Cours Initiaux (MVP)

Voici la liste des cours prévus pour le lancement MVP de l'académie.

### Cours 1 : Introduction au Web3 et à la Blockchain

*   **Objectif :** Fournir une compréhension de base de ce qu'est la blockchain, le Web3 et pourquoi c'est important.
*   **Public Cible :** Grands débutants.
*   **Modules :**
    *   **Module 1.1 : Les Fondamentaux**
        *   Leçon 1.1.1 : Qu'est-ce que le Web1, Web2, et l'avènement du Web3 ?
        *   Leçon 1.1.2 : Les problèmes que le Web3 cherche à résoudre (centralisation, données personnelles, etc.).
        *   Leçon 1.1.3 : Définition simple de la Blockchain.
    *   **Module 1.2 : Concepts Clés de la Blockchain**
        *   Leçon 1.2.1 : Blocs, Transactions, et Hashes.
        *   Leçon 1.2.2 : Décentralisation et Distribution (Ledger Distribué).
        *   Leçon 1.2.3 : Immuabilité et Transparence.
        *   Leçon 1.2.4 : Cryptographie (clés publiques/privées, signatures) expliquée simplement.
    *   **Module 1.3 : Bienvenue dans le Web3**
        *   Leçon 1.3.1 : Les piliers du Web3 (Blockchain, IA, IoT - focus blockchain).
        *   Leçon 1.3.2 : Exemples concrets d'applications Web3 (DeFi, NFTs, DAOs, Jeux).
        *   Leçon 1.3.3 : Les promesses et les défis du Web3.

### Cours 2 : Comprendre les Cryptomonnaies et les Tokens

*   **Objectif :** Expliquer ce que sont les cryptomonnaies, les tokens, et comment les gérer.
*   **Public Cible :** Débutants ayant suivi le Cours 1 ou ayant une connaissance de base.
*   **Modules :**
    *   **Module 2.1 : Les Cryptomonnaies Expliquées**
        *   Leçon 2.1.1 : Qu'est-ce qu'une cryptomonnaie ? (Ex: Bitcoin, Ethereum).
        *   Leçon 2.1.2 : Différence entre Monnaie Fiat et Cryptomonnaie.
        *   Leçon 2.1.3 : Mining et Consensus (Proof-of-Work, Proof-of-Stake - simplifié).
    *   **Module 2.2 : Les Différents Types de Tokens**
        *   Leçon 2.2.1 : Tokens Fongibles (ERC-20 comme exemple).
        *   Leçon 2.2.2 : Tokens Non-Fongibles (NFTs - ERC-721, ERC-1155 comme exemples) - Introduction.
        *   Leçon 2.2.3 : Utility Tokens vs. Security Tokens (aperçu simple).
    *   **Module 2.3 : Gérer ses Actifs Numériques (Wallets)**
        *   Leçon 2.3.1 : Qu'est-ce qu'un Wallet Crypto ? (Software, Hardware, custodial/non-custodial).
        *   Leçon 2.3.2 : Importance de la Seed Phrase et de la sécurité.
        *   Leçon 2.3.3 : Interagir avec des dApps via son Wallet.

### Cours 3 : Introduction aux NFTs (Non-Fungible Tokens)

*   **Objectif :** Approfondir la compréhension des NFTs.
*   **Public Cible :** Utilisateurs intéressés par les NFTs, artistes, créateurs.
*   **Modules :**
    *   **Module 3.1 : Qu'est-ce qu'un NFT ?**
        *   Leçon 3.1.1 : Définition détaillée et unicité.
        *   Leçon 3.1.2 : Standards courants (ERC-721, ERC-1155) et leurs différences.
        *   Leçon 3.1.3 : Métadonnées des NFTs (où sont-elles stockées ? IPFS).
    *   **Module 3.2 : Cas d'Usage et Écosystème NFT**
        *   Leçon 3.2.1 : Art numérique et objets de collection.
        *   Leçon 3.2.2 : NFTs dans les jeux (Gaming).
        *   Leçon 3.2.3 : NFTs pour les adhésions, les billets, l'immobilier (aperçu).
        *   Leçon 3.2.4 : Comprendre les Marketplaces NFT.

### Cours 4 : Démarrer avec BlockDeploy

*   **Objectif :** Présenter la plateforme BlockDeploy et comment s'y inscrire/connecter.
*   **Public Cible :** Nouveaux utilisateurs de BlockDeploy.
*   **Modules :**
    *   **Module 4.1 : Présentation de BlockDeploy**
        *   Leçon 4.1.1 : La mission de BlockDeploy : simplifier le Web3.
        *   Leçon 4.1.2 : Aperçu des principaux outils et modules.
        *   Leçon 4.1.3 : Avantages d'utiliser BlockDeploy (No-Code/Low-Code, Sécurité, Coûts).
    *   **Module 4.2 : Naviguer dans l'Interface BlockDeploy**
        *   Leçon 4.2.1 : Créer un compte / Se connecter.
        *   Leçon 4.2.2 : Le Tableau de Bord principal.
        *   Leçon 4.2.3 : Connecter son Wallet à BlockDeploy (via module WalletConnect).
        *   Leçon 4.2.4 : Paramètres du compte et support.

### Cours 5 : Créer son Premier Token avec BlockDeploy

*   **Objectif :** Guider l'utilisateur pas-à-pas pour créer un token ERC-20 avec le Token Creator.
*   **Public Cible :** Utilisateurs de BlockDeploy souhaitant créer un token.
*   **Modules :**
    *   **Module 5.1 : Prérequis et Concepts du Token Creator**
        *   Leçon 5.1.1 : Qu'est-ce qu'un token ERC-20 ? (Rappel).
        *   Leçon 5.1.2 : Options disponibles dans le Token Creator (Nom, Symbole, Supply, Décimales, Mintable, Burnable).
        *   Leçon 5.1.3 : Comprendre les frais de gas et les réseaux de test.
    *   **Module 5.2 : Tutoriel Pas-à-Pas : Créer un Token ERC-20**
        *   Leçon 5.2.1 : Accéder au Token Creator.
        *   Leçon 5.2.2 : Remplir les informations du token.
        *   Leçon 5.2.3 : Choisir les fonctionnalités optionnelles.
        *   Leçon 5.2.4 : Préparer et déployer le contrat via son Wallet (sur un Testnet).
        *   Leçon 5.2.5 : Vérifier son token sur un explorateur de blocs.
        *   Leçon 5.2.6 : (Optionnel) Interagir avec son token (le transférer).

### Cours 6 : Lancer sa Marketplace NFT avec BlockDeploy

*   **Objectif :** Guider l'utilisateur pour créer une marketplace NFT et une collection avec le NFT Marketplace Builder.
*   **Public Cible :** Utilisateurs de BlockDeploy souhaitant lancer une marketplace.
*   **Modules :**
    *   **Module 6.1 : Principes d'une Marketplace NFT**
        *   Leçon 6.1.1 : Rôle d'une marketplace et d'un contrat de collection.
        *   Leçon 6.1.2 : Options du NFT Marketplace Builder (Frais, Royalties).
    *   **Module 6.2 : Tutoriel : Configurer et Déployer votre Marketplace**
        *   Leçon 6.2.1 : Accéder au NFT Marketplace Builder.
        *   Leçon 6.2.2 : Paramétrer sa marketplace (nom, frais).
        *   Leçon 6.2.3 : Déployer le contrat de la marketplace.
    *   **Module 6.3 : Tutoriel : Créer une Collection et Lister des NFTs**
        *   Leçon 6.3.1 : Créer et déployer un contrat de collection ERC-721.
        *   Leçon 6.3.2 : Minter quelques NFTs dans sa collection.
        *   Leçon 6.3.3 : Lister un NFT à la vente sur sa marketplace.
        *   Leçon 6.3.4 : Acheter un NFT (en tant qu'autre utilisateur).

## 4. Format des Leçons

Chaque leçon sera structurée de manière claire :
*   **Titre de la Leçon.**
*   **Objectifs d'Apprentissage :** Ce que l'utilisateur saura faire après cette leçon.
*   **Contenu Principal :**
    *   Texte explicatif.
    *   Images, captures d'écran, diagrammes si nécessaire.
    *   Blocs de code (pour exemples Solidity ou JavaScript).
    *   Vidéos intégrées (si disponibles).
*   **Points Clés / Résumé.**
*   **(Optionnel) Quiz Rapide** pour auto-évaluation.
*   **(Optionnel) Ressources Supplémentaires / Liens Utiles.**
